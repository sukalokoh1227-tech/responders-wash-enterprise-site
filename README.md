# School WASH Portal Frontend

Frontend dashboard for monitoring water, sanitation, and hygiene metrics across schools. Built for Responders WASH Enterprise.

## Features

- **School Information**: View school details and contact info
- **WASH Performance**: Uptime, resilience score, and breakdown tracking
- **Issue Log**: Recent maintenance and service issues
- **Water Quality**: Live water quality test results
- **Status Indicators**: Color-coded metrics for quick assessment

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Fetch API for backend communication
